# PFT-test-page

A simple web page with a black background, white top bar, and a clickable profile dropdown menu.

## Usage

Open `index.html` in a web browser to view the page. Click the profile icon (👤) in the top right to toggle the dropdown menu with options like Settings, Profile, and Logout.

## Features

- Black background
- White top bar fixed at the top
- Profile icon on the top right
- Dropdown menu with clickable buttons

## Troubleshooting

If the page doesn't load, ensure you have a modern web browser. No dependencies required.
